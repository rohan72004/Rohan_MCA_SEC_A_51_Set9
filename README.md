## 📖 Project Description

This project is a **Gym Locker Management System** developed using Core Java and MySQL.
It helps manage gym members and locker allocation efficiently.

### 🔹 Features Implemented:

* Add new gym member
* Assign available locker to member
* Release locker when not in use
* View all lockers (occupied / available)

---

## 🗄️ Database Details

The system uses two tables:

* **members** → stores member details
* **lockers** → stores locker allocation details
